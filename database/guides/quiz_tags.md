# Quiz Tag Reference

Use these standardized tags when inserting quiz questions into `quiz_questions.tags`. Tags are lowercase, hyphenated, and refer to topic concepts â€” not lesson names.

> **Rule:** Always use the exact strings from this list. Do not create variants (e.g. use `vectors`, not `vector`).

---

## Math Module Tags

| Tag | When to use |
|-----|-------------|
| `vectors` | Any question about vectors |
| `dimension` | Questions about the number of components in a vector |
| `components` | Questions about individual entries (x-component, y-component, etc.) |
| `geometry` | Questions involving graphical/geometric interpretation |
| `addition` | Questions involving adding vectors |
| `scalar-multiplication` | Questions involving multiplying a vector by a scalar |
| `magnitude` | Questions about the length/magnitude of a vector |
| `higher-dimensions` | Questions involving vectors with more than 2 components |
| `notation` | Questions about mathematical notation or symbol conventions |
| `definition` | Questions that test vocabulary or conceptual definitions |
| `matrices` | Any question about matrices |
| `matrix-multiplication` | Questions involving multiplying matrices |
| `determinant` | Questions about the determinant of a matrix |
| `complex-numbers` | Any question about complex numbers |
| `imaginary-unit` | Questions involving `i` or imaginary unit definition |
| `complex-plane` | Questions about graphing complex numbers |
| `modulus` | Questions about the modulus (absolute value) of a complex number |
| `planar-relationships` | Questions about planes, rotations, or 2D transformations |
